# 🦖 DMOKit (Digimon Masters Online Kit)

## 📖 소개
> **DMOKit**은 디지몬 마스터즈 온라인(DMO) 플레이어들을 위한 종합 웹 유틸리티 도구입니다.
> 탐험 효율 분석부터 확률형 아이템(뽑기) 시뮬레이션, 버프 타이머까지, 스마트한 게임 플레이를 지원하는 다양한 기능을 웹에서 간편하게 제공합니다.

## ✨ 주요 기능
- **🗺️ 탐험 효율 계산기:** Lv.1 ~ Lv.170 탐험지별 경험치 효율과 필요 횟수 분석
- **🃏 데이터 드로우 시뮬레이터:** 공식 확률 기반 드로우 체험, 결과 인벤토리 누적
- **🔮 데이터 소환 시뮬레이터:** 캡슐 기반 소환, 천장(피티) 시스템 포함
- **⚗️ 데이터 융합 시뮬레이터:**
  - 동일 등급 4개 → 결과 1개 합성, 천장(피티) 시스템 포함
  - **자동 / SSS / SSS+** 모드 선택 — 자동은 하위 등급부터 순차 합성, SSS·SSS+ 모드는 해당 등급만 선별 합성
  - 10회 일괄 합성 지원
- **📊 가챠 확률 보기:** 드로우 / 소환 / 융합 각 풀의 아이템 목록과 확률 정보 조회
- **⏱️ 버프 타이머:** 버프 시간 설정 및 PiP(Picture-in-Picture) 모드로 백그라운드 추적

## 🛠 기술 스택
- **Frontend:** HTML, CSS, JavaScript (Vanilla, ES Modules)
- **Data Pipeline:** Python (원본 데이터 → JS 모듈 변환)
- **Deployment:** GitHub Pages (정적 웹 호스팅)
